### How to play:<br>
(run python game)<br>
$python3 game.py<br>
<br>
### ---------------------
press [Space Bar] for jumping and starting the game<br>
press [Left Arrow] to move character to the left<br>
press [Right Arrow] to move character to the right<br>
all [Buttons] can be holded
